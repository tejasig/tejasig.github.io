Understanding the Structural Organization of Biomolecular Assemblies Using Graph-Theoretic Techniques

Supervisor: Dr. Debayan Chakraborty, The Institute of Mathematical Sciences, Chennai

Project Overview · Code repository available upon request

This project focuses on representing biomolecular systems as graphs, where atoms or residues form nodes and interactions define edges. The goal is to study structural organization and uncover patterns that are not directly visible from raw structural data.

The work involved building Python-based pipelines for constructing graph representations, analysing connectivity, and extracting physically meaningful structural insights from biomolecular assemblies.

Anti-cancer Drug Design: Targeting ABL1

Machine Learning · Cheminformatics · Computational Drug Discovery

Independent Research Project

Won 2nd position at International Conference ‘Advances in Pharmaceutical Sciences: Bridging Academia and Industry’, St. Stephen’s College (March 2024)

GitHub Repository

This project explores computational approaches to targeting the ABL1 kinase in cancer. It combines structural understanding with data-driven techniques to study molecular interactions and inhibitor design.

The work draws on machine learning and cheminformatics to analyse molecular features and develop strategies relevant to computational drug discovery.
